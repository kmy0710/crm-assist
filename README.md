# CRM Assistant
요구사항<br>
1) 사용자가 채팅창을 통해서 '기획전에 대한 정보'를 입력한다.<br>
2) 시스템은 고객에게 발송할 CRM에 대한 '문자 메세지', '타켓 고객', '이미지"를 후보군 제안한다.<br>
3) CRM 보낼 타겟 고객군 리스트 5개와 문자메시지 문안, 이미지 프롬프트를 작성한다
4) 이미지 생성하기를 누르면 이미지 생성이 나노바나나를 통해 생성이 된다.
5) 문안 수정하기를 누르면 추가 수정할 수 있는 내용이 나온다
6) 타겟 고객 추출하기를 누르면 타겟 고객의 SQL문이 나온다
7) CRM 발송 하기를 누르면 이모든게 정리되어 '타겟 고객 SQL', '문자메시지', '이미지'가 나온다
8) 발송하고 성과분석하기를 누르면 SQL 연결된 데이터로 성과분석 결과가 나온다
